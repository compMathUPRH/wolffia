Wolffia


A graphical user interface to setup, execute, monitor and analyze molecular dynamics simulations.

What is Wolffia?

Wolffia has been developed as a tool that integrates diverse MD software to guide the user throughout the simulation process.

The design of the software is based on input from scientists with experience modeling using MD. Before starting the actual design of the GUI the following objectives and/or criteria were stated:

  • the application should be useful for novice as well as experienced CMD users,
  • given the number of options that would have to be presented to the user the use of new separate windows for a task should be avoided as they quickly clutter the workspace,
  • the interface should preserve the sense of a layered process,
  • the complexities of CMD configuration file formats would be hidden from the user unless he or she wishes to manipulate them,
  • reasonable default choices and values should be presented when no previous choice has been made,
  • the need to use the keyboard should be reduced to a minimum,
  • the molecule viewer should be sophisticated enough to give proper feedback but the emphasis of the work would not be molecule visual rendering since there is excellent software for doing that (eg, VMD, pyMol).

Why the name Wolffia?

Wolffia is the name of the genus of the smallest known flowering plant species. As the plant does, Wolffia (the application) is meant to produce beauty at a small scale.
