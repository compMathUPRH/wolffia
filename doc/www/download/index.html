Wolffia


A graphical user interface to setup, execute, monitor and analyze molecular dynamics simulations.

Downloads

Requisites

Wolffia rqeuires the following libraries and/or applications (descriptions are quotes from the products' web sites):

  • Python 2.7.1 programming language interpreter.
  • PyQt v4 set of Python bindings for Nokia's Qt application framework and the Qt 4 libraries.
  • NetworkX v4.1.6 a Python language software package for the creation, manipulation, and study of the structure, dynamics, and functions of complex networks.
  • PyBel functions and classes that make it simpler to use the Open Babel
  • Openbabel v2.3.1 chemical toolbox.
  • PyQwt - Qt Widgets for Technical Applications (v6.0).
  • PyOpenGL 3.x, a Python binding to OpenGL and related APIs.
  • NAMD, a parallel molecular dynamics code designed for high-performance simulation of large biomolecular systems.

Please don't get scared by the long list. Most of the avove libraries are part of the LINUX distributions or are included in a single Windows installation package.

Installation

The current version of Wolffia uses NAMD to run the simulations. You may build mixtures, edit force fields and other configuraton parameters and save all necessary configuration files without it. But if you plan to run simulations on the local computer you need NAMD. Therefore, for all installation options you should download and install NAMD. You should install the multicore version if the option is available for your platform. Please make sure that the executable namd2 (or a link to it) is in your PATH environment variable.

Ubuntu 12.04:

  1. Download the wolffia_0.2-1_all.deb debian package.
  2. Open a terminal and change to the folder in which the debian packaged was copied.
  3. In an account with administrator priviledges run
    sudo dpkg -i ./wolffia_0.2-1_all.deb

There are debian package managers with graphical user interfaces that you can use instead dpkg.

This debian package should work fine with other debian-based LINUX distributions but we have not tested them yet.

You may remove the package executing sudo dpkg -r wolffia.

Other Linux distributions:

  1. Install the requisite packages listed above.
  2. Download wolffia0_2.tgz.
  3. Open a terminal and change to the folder in which the tgz package was copied.
  4. In an account with administrator priviledges run
    tar -xzf wolffia0_2InstallScript
    ln -s /usr/share/wolffia/wolffia.py /usr/local/bin/wolffia
    This step will
    • copy the code to the directory /usr/share/wolffia.
    • create a link in order to be able to run wolffia by entering the command wolffia.

Windows 7:

  1. Download and install the Python(x,y) scientific and engineering development software. The default installation options don't include some of Wolffia's requisites. To make sure everything needed gets installed we suggest selecting all python packages for installation. Extra applications are not needed.
  2. Download and install OpenBabel and PyBabel.
  3. Download and execute Wolffia's installation wizard (or you may download, decompress and install wolffia0_2.zip manually).

MacOS 10.06:

Macintosheros will have to follow some extra steps. But all the installations are straightforward. We provide direct links to the installation images for Mac.
