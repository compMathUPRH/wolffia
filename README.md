# Wolffia

For python2

## Dependencies:
* python-qt
* python-qt4wol
* pyqt4-dev-tools 
* python-networkx 
* python-openbabel 
* python-qwt5-qt4 
* python-opengl
* python-qt4gl
* python-qt4-gl
* python-matplotlib
* python-scipy
* namd2
* vmd

## Installation
* git clone https://github.com/compMathUPRH/wolffia.git
* sudo cp -r path/to/wolffia /usr/local/share
* sudo ln -s  /usr/local/share/wolffia/wolffia /usr/local/bin/wolffia
